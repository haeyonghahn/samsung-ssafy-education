#### 수업목표
1. 엘리스 플랫폼과 친해지기
2. 수강생 실력 파악하여 앞으로 난이도 조절
3. 페어프로그래밍, 퀴즈 등 수업 운영방식 소개   
   

#### 수업 진행 순서   
1. (9:00~9:30) 오리엔테이션
2. (9:30~10:30) 레벨 테스트
3. (10:30~10:45) 휴식
 * 테스트 결과에 따라 조 편성
 * 편성 방식: (1, 11), (2, 12), … , (10, 20)
4. (10:45~11:00) 페어프로그래밍 소개
 * 코드공유 방법 안내
 * Default 실습방 클릭
 * 페어에게 읽기전용 권한 부여
5. (11:00~12:00) 페어프로그래밍 세션
 * 100점 받지 못했거나, 다른 풀이가 궁금한 실습문제를 공유
 * 페어로 해결하지 못 하는 문제는 선생님에게 질문
 * 시간이 남는 경우, 오후 수업 예습
